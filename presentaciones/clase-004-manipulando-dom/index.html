¿Manipular el DOM?

Cuando decimos manipular el DOM lo que hacemos es agregar, modificar y eliminar elementos a la pagina.

Para manipular el DOM tenemos métodos que podemos ejecutar pero tambien podemos hacerlo cambiando el valor de las propiedades, veamos las diferencias

Cambiando propiedades


                        /*
                            Tenemos este HTML
                            
Hola, mi nombre es Freddy

                        */

                        // Obtenemos el elemento y lo guardamos en la variable h1
                        const h1 = document.querySelector('h1');

                        // Hola, mi nombre es Freddy
                        console.log(h1.innerText);

                        h1.innerText = 'Hola, mi apellido es Montes'

                        // Hola, mi apellido es Montes
                        console.log(h1.innerText);

Ejecutando métodos


                        /*
                            Tenemos este HTML
                            
Hola, mi nombre es Freddy

                        */

                        // Obtenemos el elemento y lo guardamos en la variable h1
                        const h1 = document.querySelector('h1');

                        // Hola, mi nombre es Freddy
                        console.log(h1.innerText);

                        h1.innerText = 'Hola, mi apellido es Montes'

                        // Hola, mi apellido es Montes
                        console.log(h1.innerText);

                        h1.remove();

                        // Eliminamos el elemento de la página
Entonces, la propiedad solo le asignamos un valor y el metodo lo llamamos con ()

Agregando elementos

Demo 

                        // Creamos el elemento, todavía no existe en el DOM
                        const h1 = document.createElement('h1');
                        
                        // Le agregamos texto
                        h1.innerText = 'Hello World'
                        
                        // Ahora si lo agregamos al DOM, en este caso al body
                        document.body.appendChild(h1);
                        
                        /* 
                            Incluso despues de agregarlo y como tenemos la referecia
                            podemos cambiarle mas propiedades.
                        */
                        h1.style.color = 'red';

Esto lo hacemos en 2 pasos: creamos el elmento y lo agregamos.

  1. Crear el elemento: document.createElement
  2. Agregarlo: en este caso tenemos varios métodos
    1. appendChild y append
    2. insertBefore, insertAdjacentElement, insertAdjacentHTML, insertBefore
